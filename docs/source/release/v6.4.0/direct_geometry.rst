Improvements
############

- :ref:`DirectILLCollectData <algm-DirectILLCollectData>` will now accept fractional elastic peak reference bin as well as forward fractional elastic
  peak index to the :ref:`CorrectTOFAxis <algm-CorrectTOFAxis>`
