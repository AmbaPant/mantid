==============================
Bessel
==============================

.. index:: Bessel

Description
-----------

.. math:: 
	\begin{align} A(t)&=A_0J_0(2\pi\nu t+\phi),\\ \text{where} \qquad J_0(x)&=\sum_{l=0}^{\infty}\frac{(-1)^l}{2^{2l}(l!)^2}x^{2l} \end{align}

:math:`J_0(x)` is the Bessel Function of the first kind.
