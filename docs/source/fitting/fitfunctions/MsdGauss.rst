.. _func-MsdGauss:

========
MsdGauss
========

.. index:: MsdGauss

Description
-----------

The Gaussian model for Mean-squared displacement fitting is defined as:

.. math:: 

    \mbox{Height} \cdot e^{\left(\frac{-x^2 \cdot MSD}{6}\right)}

where:
  - :math:`MSD`  is the mean squared displacement
  - :math:`Height` is the intensity scaling
