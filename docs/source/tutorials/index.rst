=====
Training
=====

These tutorials and training courses are designed to give you a step-by-step guide to how to use some of the basic features of Mantid. The initial sections deal with some of the fundamental functions that are required for (almost) any use of of Mantid. We also include some more instrument specific training, provided by various sites.
